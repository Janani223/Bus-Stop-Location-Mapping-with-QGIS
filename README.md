# 🚌 Bus Stop Location Mapping with QGIS

## 📌 Project Overview
This project demonstrates the use of Geographic Information Systems (GIS) to map and visualize bus stop locations using QGIS. The objective is to create an accurate digital map of bus stops and understand the basics of geospatial data analysis and visualization.

## 🎯 Objectives
- Map bus stop locations using QGIS.
- Visualize public transportation infrastructure.
- Learn the basics of GIS data collection and mapping.
- Perform simple spatial analysis of bus stop locations.

## 🛠️ Software Used
- QGIS
- OpenStreetMap (Base Map)
- GeoJSON / Shapefile
- GitHub

## 📂 Project Files
- Bus Stop Dataset
- QGIS Project File (.qgz)
- Bus Stop Layer (GeoJSON/Shapefile)
- Map Layout (PNG/PDF)
- README.md

## 🔄 Methodology
1. Download the bus stop dataset.
2. Open the dataset in QGIS.
3. Add an OpenStreetMap base map.
4. Display bus stop locations.
5. Customize symbols and labels.
6. Generate the final map layout.
7. Export the map and upload the project to GitHub.

## 📊 Features
- Interactive bus stop mapping
- Spatial visualization
- Easy-to-understand map layout
- Public transport infrastructure analysis

## 🎯 Expected Output
A digital map showing the locations of bus stops with proper symbols, labels, and a professional map layout created using QGIS.
